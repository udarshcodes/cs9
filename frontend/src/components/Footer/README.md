# Footer

Site-wide footer for the Rogare platform — surfaces on all public and authenticated pages.

## Sections

- **Brand** — "rogāre" logo with a tagline about IIT Ropar
- **EXPLORE** — Contact Us, Privacy Policy, Rosetta Journal
- **INSTITUTION** — IIT Ropar Main Site, Terms of Service, Lab Guidelines

## Design

- Light gray background (`#f8f9fa`) with a top border
- Single-column on mobile, 3-column grid on desktop (`md:grid-cols-3`)
- Links have hover-to-black transition
- Copyright line at the bottom

Takes no props — hardcoded content reflects the Rogare Internship Hub brand.
